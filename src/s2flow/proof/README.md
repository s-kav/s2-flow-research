# Proof research layer

This layer supports a proof-development program for the open S²-flow conjecture.

- `lemmas/`: executable finite checks corresponding to candidate structural lemmas.
- `search/`: bounded search, candidate ordering, reductions, induction plans, and search certificates.
- `symbolic/`: optional SymPy and SOS-oriented polynomial formulations.
- `certificates/`: export, integrity checking, and independent numerical verification.

Passing these checks on finite graph families is evidence only. It is not a proof for all bridgeless cubic graphs.
