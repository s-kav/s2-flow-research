# Reproducibility

This repository separates exact constructive certificates, numerical candidate
certificates, and general mathematical claims.

## Environment

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e ".[dev,plot,report]"
```

On Windows use `.venv\Scripts\activate`.

## Test the mathematical and serialization components

```bash
pytest -q
python -m compileall -q src scripts tests
```

## Reproduce the finite campaign

```bash
python scripts/massive_verification.py \
  --orders 10 12 14 16 18 20 24 30 40 \
  --samples-per-order 30 \
  --edge-coloring-max-nodes 2000000 \
  --nonlinear-restarts 16 \
  --output-dir results/massive_run
```

The fixed seed is stored in `configs/experiments/massive.yaml` and in the
resulting `campaign.json`.

## Independently verify every stored certificate

```bash
python scripts/verify_campaign.py results/massive_run --tolerance 1e-7
```

The verifier decodes the stored graph6 graph, reconstructs the oriented
incidence matrix, and recomputes conservation residuals, row norms, and Gram
rank without trusting the solver's status.

## Stream exhaustive cubic graphs generated by nauty

```bash
geng -c -d3 -D3 16 | \
  python scripts/nauty_cubic_campaign.py --output-dir results/geng16
```

`geng` is distributed with nauty/Traces and is not bundled in this repository.

## Rebuild the Word report

```bash
python scripts/build_partial_results_report.py
```

This command reads `results/massive_run`, regenerates the figures, and writes
`docs/S2_flow_partial_theorems_and_structural_lemmas_ru.docx`.

## Interpretation

A certificate generated from a proper 3-edge-colouring is an exact
constructive result up to floating-point representation of `1/sqrt(2)`.
A nonlinear certificate is a finite numerical witness accepted under the
reported tolerance. Neither a finite campaign nor numerical convergence proves
the universal S²-flow conjecture.
